Files for creating report.pdf
=============================

report.tex  : LaTeX file, open this file using, for example MikTex's TeXworks editor

example.bib : bibliography file containing details of books, papers etc.
natbib.sty  : style files for the format of citations using \cite{} etc.
Chicago.bst : style file for the format of references
space_dep_distant.pdf : pdf file for reading into document
space_dep_nearby.pdf  : pdf file for reading into document



Files for creating latex_pres.pdf
=============================

latex_pres.tex                 : LaTeX file using beamer talk class

ucltemplate.sty                : style file
beamercolorthemelight blue.sty : style file
banners\bannerlightblue.pdf    : file containing UCL logo
latex_vs_word.pdf	       : pdf file for reading into document

UCL's Beamer LaTeX templates (including alternative colours) can be downloaded from
http://www.ucl.ac.uk/corporate-identity/logos
